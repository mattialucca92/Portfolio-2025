🚀 Come avviare il progetto in locale

Clona il repository:
git clone https://github.com/mattialucca92/Portfolio-2025.git

Accedi alla cartella del progetto:
cd Portfolio-2025

Installa le dipendenze:
npm install

Avvia il server di sviluppo:
npm run dev
